# ROSSE-HEDICKE
La Calculadora Ross-Heidecke es una herramienta técnica que estima la depreciación física de una construcción con base en su edad, vida útil y estado de conservación. Su resultado sirve como apoyo para determinar el valor neto de reposición en avalúos inmobiliarios y deberá complementarse con el criterio profesional del valuador.
